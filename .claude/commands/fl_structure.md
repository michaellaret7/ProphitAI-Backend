# Update File Structure

Generate the current project file structure and update `file_structure.md`.

## Instructions

1. Run a tree command or equivalent to get the current directory structure
2. Exclude common directories: node_modules, .git, dist, build, .next, __pycache__, venv
3. Write the output to `file_structure.md` with:
   - A timestamp of when it was generated
   - The complete directory tree
   - Brief description of main folders at the top
