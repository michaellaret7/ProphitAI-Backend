================================================================================
                        ROOT CAUSE: MISSING VALIDATION
================================================================================

The finalize tool contains NO check for task completion state.

What SHOULD happen:
  def finalize(answer):
    if not task_state.all_tasks_complete():
      raise EnforcementError("Cannot finalize with incomplete tasks")
    return success

What ACTUALLY happened:
  def finalize(answer):
    return success  # No validation, accepts any state

This single missing check allowed the entire bypass.
